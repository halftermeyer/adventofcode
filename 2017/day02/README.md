# Day 2: Corruption Checksum
[Problem](https://adventofcode.com/2017/day/2) | **Graph**: None | **CS**: Nested iteration / divisibility
## Approach
Parse each row into integers. Part 1: sum of (max - min) per row. Part 2: for each row, find the unique pair where one evenly divides the other, sum those quotients.
