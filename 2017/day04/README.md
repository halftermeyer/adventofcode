# Day 4: High-Entropy Passphrases
[Problem](https://adventofcode.com/2017/day/4) | **Graph**: None | **CS**: Set operations / anagram detection
## Approach
Part 1: a passphrase is valid if all words are distinct (size of list equals size of distinct list). Part 2: additionally, no two words may be anagrams (sort each word's letters and check distinctness).
