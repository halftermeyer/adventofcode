# Day 1: Inverse Captcha
[Problem](https://adventofcode.com/2017/day/1) | **Graph**: None | **CS**: Circular array matching
## Approach
Split input into digits. Part 1: sum digits where the next digit (circular) matches. Part 2: sum digits where the digit halfway around the circular list matches.

|------|--------|
| 1 | 1144 |
| 2 | 1194 |
