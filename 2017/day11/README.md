# Day 11: Hex Ed
[Problem](https://adventofcode.com/2017/day/11) | **Graph**: None | **CS**: Hex grid (cube coordinates)
## Approach
Use cube coordinates (x,y,z) where x+y+z=0. Each hex direction maps to a delta. Track position after all steps for Part 1 distance. Track maximum distance seen at any point for Part 2.
