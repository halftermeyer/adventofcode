# Day 3: Squares With Three Sides
[Problem](https://adventofcode.com/2016/day/3) | **Graph**: None | **CS**: Sorting / validation
## Approach
Parse each line into three integers, sort them, and check triangle inequality (a+b>c). Part 1: row-wise. Part 2: regroup by columns (groups of 3 rows, read column-wise).
