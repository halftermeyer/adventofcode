# Advent of Code 2021 in Cypher

21 days solved in pure Cypher on Neo4j 4.x/5.x with APOC and GDS.

Original repo: [github.com/halftermeyer/AoC2021WithCypher](https://github.com/halftermeyer/AoC2021WithCypher)

| Day | Graph Type | CS Concept | Approach |
|-----|-----------|------------|----------|
| 1 | Linked List | Pairwise adjacency | Sliding window via path traversal |
| 2 | Path Graph | State propagation | Sequential command processing |
| 3 | Bit Grid | Bit manipulation | Frequency counting, binary filtering |
| 4 | Grid/Board | Pattern matching | Bingo: line detection and scoring |
| 5 | Grid | Line overlay | Line segment intersection counting |
| 6 | Aggregation | Flow distribution | Lanternfish timer bin simulation |
| 7 | None | Optimization | Brute force fuel cost minimization |
| 8 | Knowledge Graph | Constraint satisfaction | Segment-to-digit mapping |
| 9 | Grid + GDS | Connected components (WCC) | Basin detection via flood fill |
| 10 | Stack Machine | Syntax validation | Bracket matching and scoring |
| 11 | Grid | Wave propagation | Cascading octopus flashes |
| 12 | Pruned DAG | Path enumeration | Cave graph path counting |
| 13 | Coordinate Set | Reflection/folding | Iterative point folding |
| 14 | Sequence Chain | Template expansion | Polymer pair insertion |
| 15 | Grid + GDS | Dijkstra shortest path | Risk-weighted grid pathfinding |
| 16 | Bit Stream | Packet parsing | Stateful bit sequence automaton |
| 17 | None | Trajectory simulation | Kinematic probe dynamics |
| 18 | Tree | Tree reduction | Nested list magnitude (explode/split) |
| 20 | Image Grid | Convolution | Neighborhood-based pixel filtering |
| 21 | State Machine | Game simulation | Deterministic dice state transitions |
| 22 | Cuboid Set | Spatial indexing | Axis-aligned cuboid union/intersection |
